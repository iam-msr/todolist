# todolist

A Simple To-Do List Web app

1) `git clone https://github.com/iam-msr/todolist.git`

2) `cd todolist/`

3) `npm i`

4) `npm start`

5) go to `http://localhost:3000/`

## NOTE: In index.js 
=> mongoose.connect("link");
=> put your own mongoDB database server link by creating one in MongoDB
